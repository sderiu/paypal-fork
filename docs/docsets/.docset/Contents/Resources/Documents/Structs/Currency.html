Currency

public struct Currency: Content, Equatable

A monetary currency for a geographical loaction, such as United States Dollar and Euro.

Model Structure
  • code

    The 3 character code that represents the currency, i.e. USD, EUR, XXX.

    Declaration

    Swift

    public let code: String
  • number

    The 3 digit numaric code for the currency. For example, USD has 840.

    Note

    If a code is less than 3 digits, it is the same as having 0s appended to it beginning, making 8 into 008.

    Declaration

    Swift

    public let number: Int
  • e

    The currency’s ‘exponent’. This is the rough expression of the relationship between the major and minor units of the currency.

    Declaration

    Swift

    public let e: Int?
  • name

    The full name of the currency, i.e. ‘United states dollar’, ‘Egyptian pound’.

    Declaration

    Swift

    public let name: String
  • init(code:)

    Gets a known currency based on its code:

    Currency(code: "USD") // Optional(Currency(code: "USD", number: 840, e: 2, name: "United States dollar"))

    This initializer is case-insensative.

    Declaration

    Swift

    public init?(code: String)

    Parameters

    code

    The 3 character code of the currency to get.

Coding
  • init(from:)

    Creates a new Currency instance based on the decoder’s contents.

    If a keyed container can be created from the decoder, the currency will be created with data from the code, number, e, and name keys; otherwise, a single value container will be created from the decoder and get an existing currency with the decoded string as its code.

    Declaration

    Swift

    public init(from decoder: Decoder)throws
  • encode(to:)

    Encodes the instance’s code to the encoders single value container.

    Declaration

    Swift

    public func encode(to encoder: Encoder)throws
